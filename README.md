# TechCart — Curso FrontEnd & React 🛒

Repositorio del alumno. Construimos una tienda en línea, **TechCart**, **clase por clase**.
Cada carpeta `clase-XX/` contiene el **código** de esa clase, sus **diapositivas** en PDF
y los **ejercicios** para practicar.

## Cómo usar este repo
1. Entra a la carpeta de la clase (por ejemplo `clase-01/`).
2. Lee su `README.md`: qué aprendiste y cómo ver la página.
3. Abre el código y resuelve los **ejercicios** (están dentro del propio código).
4. Sube tus avances a tu GitHub con `git add .` → `git commit` → `git push`.

## Herramientas
- **Visual Studio Code** — el editor ([code.visualstudio.com](https://code.visualstudio.com)).
- **Un navegador** — Chrome o Edge.
- **Git + cuenta de GitHub** — para subir tu trabajo.
- Extensión recomendada: **Live Server**.

> Node.js y frameworks se usan más adelante (desde la Semana 3 y en React). En las
> primeras clases basta con abrir los archivos `.html` en el navegador.
