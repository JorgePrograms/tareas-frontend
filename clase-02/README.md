# Clase 2 — Introducción a CSS

> 🔜 **Próxima clase.** Aquí encontrarás el código y las diapositivas después de la sesión.
> Por ahora, estos son los temas que veremos.

En la Clase 1 dejamos la tienda "sin diseño". En la Clase 2 le damos sus **primeros
estilos** y aprendemos las bases de CSS.

## Temas a tocar
1. **Qué es CSS y cómo se aplica** — estilos en línea, `<style>` interno y **hoja externa** (`<link>`, la forma recomendada). Anatomía de una regla: `selector { propiedad: valor; }`.
2. **Selectores** — de etiqueta, de clase, de id, descendientes y de estado (`:hover`).
3. **Especificidad** — por qué una regla le gana a otra (id > clase > etiqueta).
4. **Modelo de cajas (box model)** — `content`, `padding`, `border`, `margin`; `box-sizing`; centrar con `margin: 0 auto`.

## Qué construiremos
Le daremos estilo al `index.html` de la Clase 1: colores, tipografía y las tarjetas de
producto empezarán a verse como una tienda de verdad.
